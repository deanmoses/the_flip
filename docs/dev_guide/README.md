This folder contains the developer guide.

AI assistants must follow these guides before responding:

1. If writing HTML or CSS: `docs/dev_guide/HTML_CSS_Guide.md` – enforce tokens, responsive rules, and component patterns.
2. If writing Python: `docs/dev_guide/Django_Python_Guide.md` – follow the Django/Python conventions, organization, and testing expectations.
3. If writing tests: `docs/dev_guide/Testing_Guide.md` – follow the standardized test runner, layout, and coverage expectations before writing or updating tests.
4. `docs/dev_guide/Project_Structure.md` – keep contributions aligned with the target app layout.
5. `docs/dev_guide/Data_Model.md` – respect the domain objects, fields, and privacy constraints.

AI assistants must confirm relevant sections from these documents before generating code or explanations.  Cite which guide sections informed the answer whenever practical.
